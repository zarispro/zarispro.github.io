Windows 11

     Windows 11是美国微软公司研发的一套操作系统，于2021年6月24日正式发布，于10月5日正式上市。Windows 11的外观设计与前代相比更加现代化，包括菜单“圆角化”，任务栏图标居中等。微软还重新设计了开始菜单、应用程序商店和设置应用程序，在屏幕上排列多个应用程序窗口变得更容易。
     微软简化Windows用户界面，带来了新的Windows商店，以及对性能和多任务的改进。Windows 11总体设计风格正在无限接近Mac OS以及Chrome OS。居中显示的外观以及圆角UI成为主要布局和设计风格。
     Windows 11在平板上做了不少改进，当然居中的图标设计让你联想到iPad，发布会上，微软展示了Windows 11直接投屏到电视上的功能，但是兼容的产品并未详细说明。
     Windows 11的Gaming游戏功能重点展示了支持Auto HDR功能，以及自动为游戏设置HDR模式改善画质的功能。
     此外，XBOX上的游戏可以通过XBOX APP集成到Windows 11中，反过来也可以在Windows 11上买XBOX游戏。
     Windows 11重做了微软的应用商店，微软暂时不会通过应用商店向开发者抽成，这是直接针对苹果的策略。应用商店还可以运行安卓应用。微软用实际行动证明，在靠近苹果的同时准备“弯道超车”。
     Windows 11具有全新的用户界面，半透明、阴影和圆角在整个系统中都很普遍。“开始”菜单被重新设计，取消了右侧的磁贴。任务栏也被简化，在默认状态下被居中，类似于MacOS，但仍保留左对齐选项。任务视图是Windows 10中引入的一项功能，在Windows 11中具有全新的设计。系统的其他更改包括新的系统图标、声音和小部件等。大部分界面和开始菜单都从现已取消的Windows 10X中汲取了许多灵感。
     Windows 11使用了新字体Segoe UI Variable。该字体对于高DPI的显示器的显示效果更好，而旧的Segoe UI没有考虑到这一点。
     Windows 11中包含一个Widget面板，Widget的图标将默认在开始菜单和Windows图标旁的任务栏中，点击后将显示所在地区的天气和新闻、交通状况、股市行情等信息。在泄露的测试版本中，无法拖动或重新排列小部件的内容，访问该面板需要使用Microsoft帐户登录。日前从泄漏出来的Windows 11接口上看来，Widget似乎是重新设计版的“新闻和兴趣”。不过微软正考虑是否支持第三方Widget，允许用户选择适合自己的Widget来自定义桌面。最初Windows 11的API仅限于第一方 Widget，但今年稍晚或明年将增加对第三方 Widget的支持。
     在Windows 11中微软改善了音量控制窗口，因为在Windows 10音量弹出窗口不是很好用。开发者Ahmed Walid在Twitter上透露了一张新版本Windows 11的弹出音量控件图片，并给出了该图片的来源路径。据消息显示，该控件似乎同样源于原本为双屏和平板设备研发的Windows 10X，是C-Shell的一部分。
     新的应用程序商店Microsoft Store，让用户可以下载更多元的App。微软强调，所有Android App都可以在Windows 11运行。另外，微软变更收益共享政策，亦即应用程序开发人员可以将自有商务导入Microsoft Store，获得100%收益，微软将不共享收益；或者，应用程序开发人员仍可选择使用微软提供的商务系统，并以85:15的比例共享收益。这项大开放，是发布会一大亮点。
     Windows 11的开机音效被更改。
